  "text": "Свойства контейнера - находятся в обьекте `props`;\r\n\r\nВ зависимости от типа здесь могут присутствовать разные поля.\r\n\r\nПоля для всех типов свойств:\r\n\r\n* `type`- тип свойства в данном случае это \"click\";\r\n* `htmlLink` - html ссылка на данное свойство;\r\n\r\nЕсли свойство является событием, группой или вариантом(render-variant):\r\n\r\n* `propName` - название свойства;\r\n* `parent`- доступ к контейнеру из данного свойства;\r\n* `pathToCоmponent` - имя компонента;\r\n* `prop` - какие либо произвольные данные, по умолчанию null;\r\n* `rootLink` - корневая ссылка на экземпляр приложения HtmlixState;\r\n\r\nДополнительные поля для типа `group`:\r\n* `groupChild` - массив с контейнерами данной группы;\r\n* `groupArray` - ссылка на виртуальный массив контейнеров данной группы;\r\n\r\nДополнительное поле для типа `render-variant`:\r\n* `renderChild`- ссылка на отображаемый компонент в данном свойстве, либо на контейнер виртуального массива. \r\n\r\nДополнительные поля для пользовательских событий:\r\n\r\n* `emiter` доступ к пользовательскому событию из свойства;\r\n* `emiterKey` уникальный ключ слушателя пользовательского события;\r\n\r\n\r\nДополнительное поле для стандартных событий:\r\n* `events` - объект с методами обработчиками событий для свойства, доступ по имени события;\r\n\r\nДополнительное поле для свойства с типом data:\r\n* `parent` - доступ к контейнеру из данного свойства;\r\n* `propName` - название свойства;\r\n"
