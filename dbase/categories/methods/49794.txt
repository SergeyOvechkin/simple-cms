  "text": "\r\nМетоды экземпляра приложения можно вызвать из любой точки this.rootLink.nameMethod();\r\n\r\n* `.getDifrentFilds(array, fild)` - получает разные поля из массива array - если массив содержит объекты, то можно указать поле объекта fild, таким образом можно \"отсеять\" повторяющиеся поля и получить только разные;\r\n\r\n* `.addContainer(stateNameProp, properties, insertLocation)` добавляет новый контейнер в массив  stateNameProp с начальными свойствами properties, в указанную позицию insertLocation,\r\nесли не указать позицию, добавит - в конец, если не указать какое либо свойство в объекте  properties то возьмет данные из шаблона.\r\n\r\n* `.removeAll(stateNameProp, widthChild)` - очищает массив stateNameProp, если указать widthChild=true удалит все дочерние компоненты со свойств с типами group и render-variant\r\n\r\n* `.removeByIndexes(stateNameProp, indexArray, widthChild )` - удаляет несколько контейнеров indexArray=[] из массива  stateNameProp;\r\n\r\n* `.removeByIndex(stateNameProp, index, widthChild )` - удаляет один контейнер из массива stateNameProp по индексу index;\r\n\r\n* `changeOrder(stateNameProp, newOrder=array)` - изменяет последовательность контейнеров  в массиве stateNameProp и html разметке, newOrder- массив со старыми индексами в новом порядке например [3,2,1,0]\r\n\r\n"
