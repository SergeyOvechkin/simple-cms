  "text": "\r\nСуществует два способа указания свойств:\r\n\r\n* 1 - Указываем свойство в html разметке с помощью `data-container_name-prop_name`, либо `data-array_name-prop_name` - для свойст массива. \r\nДалее в описании приложения указываем его в массиве props: `[\"prop_name\", ...]`, либо `arrayProps: [\"prop_name\", ...]`- для свойств массива.\r\n\r\n* 2 - Указываем свойство только в описании приложения с помощью массива `props:[ [\"prop_name\", \"prop_type\", \"selector\"]  ]`.\r\nгде \"selector\" - селектор для поиска свойства относительно контейнера, либо массива - для свойств массива, например `\"a:first-of-type\"`. \r\nЕсли селектор указать пустым `\"\"` это будет означать что свойство является в том-же теге что и контейнер, либо массив - для свойств массива.\r\n\r\n\r\nИз примера выше (# Прослушивание пользовательских событий из свойств массива) свойства можно указать так:\r\n\r\n* для свойств массива: `arrayProps:[ [\"class_menu\", \"class\", \"\"], [ \"listener_load_page\", \"emiter-load-page\", \"\" ] ]` - тотже тег что и у массива;\r\n\r\n* для свойств контейнера: `props:[ [\"class_item\", \"class\", \"\"], [\"text_item\", \"text\", \"a:first-of-type\"] ]`  a:first-of-type - селектор относительно контейнера\r\n\r\nТеперь в html разметке у нас останется только массив и два контейнера:\r\n\r\n```html\r\n\r\n\t\t<div data-menu=\"array\" >\r\n\t\t\r\n\t\t\t<div data-item=\"container\" >\r\n\t\t\t  <a > текст 1 </a>\r\n\t\t\t</div>\r\n\t\t\t\r\n\t\t\t<div data-item=\"container\" >\r\n\t\t\t  <a > текст 2 </a>\r\n\t\t\t</div>\r\n\t\t\t\r\n\t   </div>\t\t\r\n\r\n```"
