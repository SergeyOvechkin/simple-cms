  "text": "Виртуальный массив это массив в котором нет ссылки на html тег.\r\nОн нужен для создания контейнеров не сгрупированных в одном html элементе а разбросанных по разным свойствам `group` разных контейнеров.\r\n\r\nНапример у нас есть три контейнера и в каждом есть свойство с типом group\r\n\r\n```html\r\n<div data-pages=array>\r\n\r\n\t<div data-page=\"container\">\r\n\t\t<div data-page-some_group=\"group\">\r\n\t\t</div>\r\n\t</div>\r\n\t<div data-page=\"container\">\r\n\t\t<div data-page-some_group=\"group\">\r\n\t\t</div>\r\n\t</div>\r\n\t<div data-page=\"container\">\r\n\t\t<div data-page-some_group=\"group\">\r\n\t\t</div>\r\n\t</div>\t\r\n</div> \r\n\r\n\r\n```\r\n\r\n```javscript\r\n\t\r\n\tvar StateMap ={\r\n\t\t\tpages: {\r\n\t\t\t\tcontainer: \"page\",\r\n\t\t\t\tprops: [\"some_group\"],\r\n\t\t\t\tmethods: {\r\n\t\t\t\t\r\n\t\t\t\t\r\n\t\t\t\t}\t\t\t\r\n\t\t\t}\t\t\t\t\r\n\t}\r\n\r\n```\r\n\r\nТеперь в каждом свойстве `some_group` мы хотим поместить различное количество пунктов меню data-item=\"container\" например в первом 1, во втором 2, в третьем 3\r\nесли использовать обычные массивы то нам прийдется создать три одинаковых массива с различным набором data-item=\"container\" и одинаковой функциональностью.\r\nЧтобы этого не делать мы создадим один виртуальный массив items а его контейнеры мы распределим между свойствами \"some_group\"\r\n\r\nИтак html код будет выглядеть так:\r\n\r\n```html\r\n<div data-pages=array>\r\n\r\n\t<div data-page=\"container\">\r\n\t\t<div data-page-some_group=\"group\">\r\n\t\t\r\n\t\t\t<div data-item=\"container\" data-item-text=\"text\">текст 1</div> <!-- добавили контейнер item -->\r\n\t\t\t\r\n\t\t</div>\r\n\t</div>\r\n\t<div data-page=\"container\">\r\n\t\t<div data-page-some_group=\"group\">\r\n\t\t\r\n\t\t\t<div data-item=\"container\" data-item-text=\"text\">текст 1</div>\r\n\t\t\t<div data-item=\"container\" data-item-text=\"text\">текст 2</div>\r\n\t\t\r\n\t\t</div>\r\n\t</div>\r\n\t<div data-page=\"container\">\r\n\t\t<div data-page-some_group=\"group\">\r\n\t\t\r\n\t\t\t<div data-item=\"container\" data-item-text=\"text\">текст 1</div>\r\n\t\t\t<div data-item=\"container\" data-item-text=\"text\">текст 2</div>\r\n\t\t\t<div data-item=\"container\" data-item-text=\"text\">текст 3</div>\t\t\t\r\n\t\t\r\n\t\t</div>\r\n\t</div>\t\r\n</div> \r\n\r\n\r\n```\r\n\r\nТеперь добавим виртуальный массив items в javascript код: \r\n\r\n```javscript\r\n\t\r\n\tvar StateMap ={\r\n\t\t\tpages: {\r\n\t\t\t\tcontainer: \"page\",\r\n\t\t\t\tprops: [\"some_group\"],\r\n\t\t\t\tmethods: {\r\n\t\t\t\t\r\n\t\t\t\t\r\n\t\t\t\t}\t\t\t\r\n\t\t\t},\r\n\t\t\t\r\n\t\tvirtualArrayComponents: {\t//объект для хранения виртуальных массивов\r\n\t\t\r\n\t\t\titems: { //виртуальный массив\r\n\t\t\t\tcontainer: \"item\", //контейнер виртуального массива\r\n\t\t\t\tprops: [\"text\"],\r\n\t\t\t\tmethods: {\r\n\t\t\t\t\r\n\t\t\t\t\r\n\t\t\t\t}\t\t\t\r\n\t\t\t}\r\n\t\t}\t\r\n\t\t\t\r\n\t}\r\n\r\n```\r\n\r\nТеперь открыв в консоли экземпляр приложения можно увидеть что всего в массиве state.items.data у нас 6 контейнеров \"item\",\r\nа в свойстве  some_group.groupChild первого контейнера \"page\" - один, второго - два, третьего - три.\r\n\r\nТаким образом с помощью свойства с типом `group` мы сгрупировали в трех контейнерах \"page\" массива \"pages\" различное количество контейнеров \"item\" из массива 'items'\r\n\r\nЗаметьте что index контейнера в массиве отличается от индекса контейнера в группе groupId\r\nТеперь если мы захотим удалить контейнер из группы, с помощью метода .removeFromGroup(groupID) то он также удалится из виртуального массива.\r\n\r\n**Важно** размещать контейнеры в html разметке непосредственно в теге с типом `group` избегая каких либо промежуточных тегов, т.к. при инициализации приложение будет искать только непосредственных потомков (`this.htmlLink.children`) с типом `container`, И если их обернуть в другой тег, то оно их просто не найдет. \r\n"
